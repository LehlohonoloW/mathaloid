# Overview

MATHALOID is a lightweight, browser-only learning game for practicing arithmetic.

Key features:
- Adaptive difficulty, scoring, and streaks
- Achievements and level progression
- Problem explanations for learning
- Planned modes: Speed, Puzzle, Story
- Themes including Cockpit Neon: HUD overlays (status lights, mini radar), scanlines, and mechanical bezel

Directory (planned stable naming):
- `index.html` — UI scaffold
- `styles.css` — styles
- `mathbot.js` — core logic class (unified)
- `app.js` — UI glue, event handlers, modes
- `docs/deepwiki/` — operational docs
