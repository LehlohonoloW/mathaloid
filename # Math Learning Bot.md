# Math Learning Bot

An interactive bot that helps you improve your math skills.

## Features

- Basic arithmetic operations
- Practice exercises
- Progress tracking

## Project Structure

- `src/`: Source code containing math logic and bot interface
- `tests/`: Test cases for math operations
